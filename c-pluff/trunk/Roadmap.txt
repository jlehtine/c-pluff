C-Pluff (0.1.2) alpha

  * Stop plug-ins within a dependency loop in reversed startup order.
  * Install documentation as part of installation.
  * Install example source code as part of installation.
  * Respect DESTDIR during installation.
  * Allow special installation prefixes for examples
    (--with-examples-bindir, --with-examples-libdir).
