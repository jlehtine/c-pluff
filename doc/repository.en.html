Repository

Git is used as the version control system for C-Pluff source code, documentation and web site material. The main repository is published at GitHub as jlehtine / c‑pluff. 

git clone git@github.com:jlehtine/c-pluff.git

This is a live repository being updated in real time as new changes are committed. The source code in the master branch is work in progress. See release tags for released versions.
